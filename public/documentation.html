NetZeroFlix

This API that I have built will interact with a database and present information about movies related to the fight against climate change

The information presented will include a description of the film, the genre and the director.

Endpoint Documentation:

Request URL HTTP Method Response
Return a list of all movies to the user /movies GET A JSON object holding data about all movies
Return data (description, genre, director, image URL, whether it’s featured or not) about a single movie by title to the user /movies/:title GET a JSON object holding data about a single movie containing the data above
Return data about a genre (description) by name/title (e.g., “Thriller”) /movies/genres/:genreName GET JSON object holding data about genre
Return data about a director (bio, birth year, death year) by name /movies/directors/:directorName GET JSON object holding data about director
Allow new users to register /users POST a text message indicating successfull registration of the account
Allow users to update their user info (username) /users/:id PUT a text saying the info got updated
Allow users to add a movie to their list of favorites (showing only a text that a movie has been added—more on this later) /users/:id/:movieTitle POST a text message that movie was added to favorite list
Allow users to remove a movie from their list of favorites (showing only a text that a movie has been removed—more on this later) /users/:id/:movieTitle DELETE text message that movie was removed from favorites list
Allow existing users to deregister (showing only a text that a user email has been removed—more on this later) /users/:id DELETE a text message indicating successfull deletion of the user account
